# Features

| Playback | Audio |
|---|---|
| Stream songs & videos from YouTube Music | Audio normalization |
| Background playback | Tempo & pitch control |
| Offline downloads & caching | Equalizer |
| Skip silence | High quality streaming |
| Sleep timer | Material You audio UI |

| Lyrics & Discovery | Library & Account |
|---|---|
| Live synced lyrics | Playlist management |
| AI lyrics translation | Import playlists |
| Personalized quick picks | Queue reordering |
| Smart search | YouTube Music login |
| Discover artists & albums | Library sync |

| Interface | Social |
|---|---|
| Material 3 design | Listen together |
| Dynamic themes | Share songs easily |
| AMOLED dark mode | Discord Rich Presence |
| Multiple color palettes | Real-time sessions |
